## Project Overview
This project analyzes potential bias in resume shortlisting based on college tier.
## Tools Used
- MySQL
- Power BI Desktop
- DAX
## Key Metrics
- Total Applications by College Tier
- Shortlist Rate (%) by College Tier
## Key Insights
- Tier-1 candidates show 100% shortlist rate.
- Tier-2 candidates show 50% shortlist rate.
- Indicates potential college-tier bias in hiring decisions.
